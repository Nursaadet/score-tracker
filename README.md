# 🎯 Score Tracker Challenge

Score Tracker is a simple two-player score tracking application. Players can increase their scores by clicking their respective buttons. The game ends when one of the players reaches the predefined winning score, and the winner is visually highlighted.

## 🧠 Learning Outcomes

Through this project, the following concepts were practiced:

- **DOM Manipulation**  
  Dynamically updating and modifying HTML elements using JavaScript

- **Event Handling**  
  Handling user interactions such as button clicks and input changes

- **Variables & State Management**  
  Managing application state, including player scores and winning score

- **Conditional Statements**  
  Controlling program flow by checking conditions (e.g., determining the winner)

- **Button & Input Handling**  
  Working with buttons and input elements, and attaching event listeners

- **Dynamic Styling with JavaScript**  
  Applying style changes based on game state (winner/loser highlighting)

## 📌 Features

- **Player 1 Button**  
  Increases Player 1’s score

- **Player 2 Button**  
  Increases Player 2’s score

- **Winning Score Input**  
  Allows users to set a custom winning score

- **Reset Button**  
  Resets both players' scores and starts a new game

- **Visual Feedback**  
  - Winning player's score turns **green**
  - Losing player's score turns **red**
